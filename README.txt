HOW TO USE WITH GITHUB PAGES
1. Create a new GitHub repository.
2. Upload index.html to the main/root folder.
3. Open Settings > Pages.
4. Under Build and deployment, choose "Deploy from a branch."
5. Choose the main branch and /root, then Save.
6. Use the GitHub Pages URL inside your 3D modeler's web-view/embed object.

EDIT VIDEO NAMES
Open index.html and find:
const videos = [...]

Change "Video 1", "Video 2", etc. to the titles you want.
Do not change the id values unless replacing a video.
